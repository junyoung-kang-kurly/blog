## 환영합니다!

이것은 Astro + Tailwind CSS + MDX로 만든 기술 블로그의 첫 번째 포스트입니다.

### 블로그 기능 소개

이 블로그는 다음과 같은 기능들을 제공합니다:

1. **포스트 검색** - 키보드에서 `/` 키를 눌러 검색 모달을 열 수 있습니다
2. **태그 시스템** - 포스트를 태그별로 분류하고 탐색할 수 있습니다
3. **연관 글** - 같은 태그를 공유하는 관련 포스트를 자동으로 표시합니다
4. **다크 모드** - 시스템 설정을 따르거나 수동으로 전환할 수 있습니다
5. **목차 (TOC)** - 긴 글의 경우 오른쪽에 목차가 표시됩니다
6. **코드 하이라이팅** - 아름다운 문법 강조 기능

### 코드 예제

JavaScript 코드 예제입니다:

```javascript
// Astro에서 콘텐츠 컬렉션 사용하기
import { getCollection } from 'astro:content';

const posts = await getCollection('blog');
const sortedPosts = posts.sort(
  (a, b) => b.data.pubDate.valueOf() - a.data.pubDate.valueOf()
);
```

TypeScript도 지원합니다:

```typescript
interface Post {
  title: string;
  description: string;
  pubDate: Date;
  tags: string[];
}

function formatPost(post: Post): string {
  return `${post.title} - ${post.description}`;
}
```
