코드 위스퍼러는 깃헙의 코파일럿 같은건데, 코파일럿 쓰기전에 체험해보기 위해 개인용으로 무료이고 Intellij 깔아보기로 했다.

참고 : https://aws.amazon.com/ko/blogs/aws/amazon-codewhisperer-free-for-individual-use-is-now-generally-available/





## 시작

### 1. AWS Builder id 생성 
개발자 아이디 같은건가 보다.
개인 무료버전의 `코드 위스퍼러`는 AWS Builder id로만 인증이 가능하다. 
[가이드 문서](https://docs.aws.amazon.com/signin/latest/userguide/create-aws_builder_id.html) 에서, `To create your AWS Builder ID` 섹션을 따라하면 된다. 
1분 이면 만들수 있다.

### 2. Intellij 플러그인 설치
[AWS toolkit 플러그인](https://plugins.jetbrains.com/plugin/11349-aws-toolkit) 설치하면 된다.

해당 플러그인은 s3 익스플로러 등 AWS 서비스를 쉽게 접근하기 위한 피처들이 포함 되어 있는데, 이번에 추가된 `코드 위스퍼러` 도 포함되어 있다

설치후 재시작 하자. 

#### AWS Toolkit 패널 접근

View -> Tool Windows -> AWS Toolkit 을 클릭하도록 한다.
![](/images/posts/d98cf847-bdf8-4d43-b0c4-ccfbec2a0e30-image.png)

다음과 같은 패널을 띄울수 있고, Developer Tools 에 CodeWhispereer 탭이 보인다면 성공한 것이다.
(참고로 나는 aws cli 인증을 로컬에 해둔 상태라 약간 다르게 보여질수는 있다)
