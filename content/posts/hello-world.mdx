## 안녕하세요!

이 블로그는 **Next.js 15**와 **Velite**를 활용하여 만든 정적 기술 블로그입니다.

### 사용된 기술 스택

- **Next.js 15** - App Router를 사용한 최신 React 프레임워크
- **Velite** - 타입 안전한 MDX 콘텐츠 관리
- **Tailwind CSS** - 유틸리티 기반의 CSS 프레임워크
- **Rehype Pretty Code** - 코드 하이라이팅

### 코드 예시

다음은 간단한 React 컴포넌트 예시입니다:

```tsx
function Welcome({ name }: { name: string }) {
  return <h1>안녕하세요, {name}님!</h1>;
}

export default function App() {
  return <Welcome name="개발자" />;
}
```
