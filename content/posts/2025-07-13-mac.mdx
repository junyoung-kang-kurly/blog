요즘 이런저런 키 발급이 많아지면서 여러 컴퓨터에서 관리하는데 어려움이 있다.
환경 변수를 zprofile과 icloud를 이용해 여러 컴퓨터에서 동기화 하는법을 알아보자.


> (참고) zprofile 과 zshrc 의 차이 : https://www.reddit.com/r/zsh/comments/e882c4/what_is_the_difference_between_zshrc_and_zprofile/?tl=ko


## zprofile에 환경변수 추가

편집 및 반영
```
vi ~/.zprofile # 편집
source ~/.zprofile # 반영
``` 
![](/images/posts/b0dc63ab-e80d-474d-b46b-2d3460ace247-image.png)



잘 반영됏는지 확인
```
echo $NOTION_API_KEY
```



## 심볼릭 링크

로컬의 `.zprofile` 을 클라우드로 이동

```
mv ~/.zprofile ~/Library/Mobile\ Documents/com~apple~CloudDocs/dots/
```

클라우드의 파일을 로컬로 링크
```
ln -s ~/Library/Mobile\ Documents/com~apple~CloudDocs/dots/.zprofile ~/.zprofile
```
