## STDIO 


jar 파일 실행 예시 : 

```
    "my-mcp-calculator": {
      "command": "java",
      "args": [
        "-jar",
        "/Users/bonjugi/IdeaProjects/my-mcp-servers/my-mcp-calculator/build/libs/my-mcp-calculator-1.0.0.jar",
        "--spring.profiles.active=stdio"
      ]
    }
```    


docker 파일 실행 예시 : 

```
    "my-mcp-calculator": {
      "command": "docker",
      "args": [
        "run",
        "-i",
        "--rm",
        "bonjugi/my-mcp-calculator:latest",
        "--spring.profiles.active=stdio"
      ]
    }


```
