## mas란?

**mas**(Mac App Store CLI)는 App Store 앱을 터미널에서 설치하고 관리할 수 있는 도구다.
GUI 없이도 검색, 설치, 업데이트가 가능하다.

### 주요 기능

- Mac App Store 앱 검색
- 명령어로 앱 설치/제거
- 업데이트 관리
- Homebrew와 함께 사용

## 설치 및 기본 사용법

### 1. mas 설치

Homebrew로 설치하면 끝이다.

```bash
brew install mas
```

### 2. 카카오톡 설치

App Store에서 앱 ID를 찾아 설치한다.

```bash
# 앱 검색
mas search KakaoTalk

# 출력 예시:
# 869223134  KakaoTalk (2.9.3)

# 앱 설치 (ID 사용)
mas install 869223134
```

버전 번호는 시점에 따라 달라질 수 있다.

## 주요 명령어

### 설치된 앱 관리

```bash
# 현재 설치된 앱 목록 확인
mas list

# 특정 앱 제거
mas uninstall [APP_ID]
```

### 업데이트 관리

```bash
# 업데이트 가능한 앱 확인
mas outdated

# 모든 앱 업데이트
mas upgrade

# 특정 앱만 업데이트
mas upgrade [APP_ID]
```

### 계정 관리

```bash
# 현재 로그인된 Apple ID 확인
mas account

# 로그아웃
mas signout
```

## 문제 해결

### 로그인 문제

`missing argument for apple ID` 에러가 뜨는 경우가 있다.
이럴 땐 아래 순서로 해결하는 게 가장 안정적이다.

```bash
# 1. App Store 앱에서 수동 로그인
# GUI로 App Store 앱을 열고 로그인

# 2. mas 초기화
mas reset

# 3. 다시 시도
mas install [APP_ID]
```

**주의사항**
- `mas signin` 명령은 macOS 10.13+ 및 2단계 인증 환경에서 동작하지 않을 수 있다.
- App Store 앱에 먼저 로그인한 뒤 mas를 쓰는 것이 안전하다.

## Brewfile과 함께 사용하기

mas를 Brewfile에 넣어두면 개발 환경을 자동화하기 훨씬 좋다.

```ruby
# Brewfile 예시

# CLI 도구
brew "git"
brew "mas"

# GUI 애플리케이션
cask "visual-studio-code"
cask "google-chrome"

# Mac App Store 앱
mas "KakaoTalk", id: 869223134
mas "Magnet", id: 441258766
mas "Xcode", id: 497799835
```

### Brewfile 사용법

```bash
# 현재 설치된 패키지/앱 백업 (Brewfile 생성)
brew bundle dump

# 새 Mac에서 Brewfile로 환경 구성
brew bundle install

# 누락된 항목 확인
brew bundle check

# Brewfile에 없는 항목 정리
brew bundle cleanup --force
```

이렇게 하면:
- 여러 디바이스에서 동일한 환경 유지
- 새 Mac 설정 시간 단축
- 개발 환경을 코드로 관리
- 팀 환경 공유가 쉬움

## 추가 팁

### 1. 앱 ID 빠르게 찾기

```bash
# 앱 이름으로 검색해 ID만 추출
mas search "Xcode" | awk '{print $1}'
```

### 2. 복수 앱 일괄 설치

```bash
# 여러 앱 한 번에 설치
mas install 869223134 441258766 497799835
```

### 3. 설치 스크립트 예시

```bash
#!/bin/bash
# mac-setup.sh

echo "Installing Homebrew packages..."
brew install git mas

echo "Installing Mac App Store apps..."
mas install 869223134  # KakaoTalk
mas install 441258766  # Magnet

echo "Setup complete!"
```
